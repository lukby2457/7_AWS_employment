DOM

Document Object Model

  • 문서 객체 모델
  • 웹 문서의 모든 요소를 JS로 처리할 수 있게 만들어둔 객체

DOM으로 요소를 찾아내는 여러가지 방법

종류 설명
getElementById id 선택장로 요소를 접근
getElementsByClassName class 선택자로 요소를 접근
getElementsByName 태그의 name="" 속성으로 요소를 접근
getElementsByTagName 태그 선택자로 요소를 접근
querySelector 다양한(ID,NAME,TAGNAME) 방법으로 요소를 접근 (한개만 접근)
querySelectorAll 다양한 방법으로 요소를 접근 (여러개 접근)

ID : 고유속성이라 중복되면 위의 것만 출력된다

Class

Class : 반환값이 HTMLCollection

Name : 반환값이 NodeList. 태그의 이름을 찾는 것이 아니다.

TagName
