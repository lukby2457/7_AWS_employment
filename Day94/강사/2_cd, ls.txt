1. ls (list)
- 지정 대상 디렉토리의 목록을 확인

명령어)
ls [옵션] [대상]
- 옵션은 생략 가능
- 대상은 생략시 현재 위치(= pwd)내를 확인

옵션)
-l	: long, 상세 내용을 출력
-a	: all, 숨김파일(.파일)까지 모두 출력
-R	: Recursive, 하위 디렉토리 목록도 모두 확인

drwxr-xr-x. 2 root root 4096 11월 25 00:11 바탕화면
①    ②      ③   ④    ⑤    ⑥         ⑦        ⑧

① 파일 유형 : -는 일반파일, d는 디렉토리
② 파일 권한 : r(read), w(write), x(excute)
③ 하드 링크 : 동일한 파일의 개수
④ 파일 소유자 : 파일의 주인
⑤ 파일 소유그룹 : 파일의 그룹
⑥ 파일의 크기 : byte 단위
⑦ 파일의 날짜 : 생성 및 수정 날짜
⑧ 파일의 이름


ex)
1. ls		: 현재 디렉토리 목록을 확인
2. ls [대상]	: 지정 디렉토리의 목록을 확인
3. ls -l	: 현재 디렉토리를 상세하게 확인
4. ls -a	: 숨김 파일까지 확인
5. ls -R	: 하위 디렉토리 목록도 확인
6. ls -l /	: / 위치를 상세하게 확인
7. ls -l -a	: 현재 디렉토리 모든 파일을 상세하게
8. ls -la	: 명령어는 조합이 가능 순서 무관



※ 경로(path)
- 파일이 있는 위치를 의미
1. 절대경로 : 경로를 명시할 때 /부터 명시하는 것
2. 상대경로 : 경로를 명시할 때 자신의 위치부터 시작

※ 자신의 위치랑 가까우면 상대경로가 유리하다


※ man 명령어 : 메뉴얼을 출력하는 명령
- ex) man ls : ls의 관한 메뉴얼을 출력
- enter : 한 줄씩 다음으로
- b : 한 페이지씩 이전으로
- space : 한 페이지씩 다음으로
- q : 종료



2. cd (Change Directory)
- 디렉토리를 변경하는 명령어

사용법)
cd [대상]

경로 축약 문자)
.		현재 디렉토리
..		상위 디렉토리
~		홈 디렉토리
-		이전 디렉토리
~[계정명]		지정 계정의 홈디렉토리
