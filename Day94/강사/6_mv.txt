1. mv (move)
- 파일이나 디렉토리를 이동
- 파일이나 디렉토리 이름 변경

명령어)
mv [원본] [대상]
- 원본에서 대상 아래로 이동 시킨다
- 대상 작성시 디렉토리까지만 작성하면 원본명으로 이동
- 대상 디렉토리 아래 이름을 작성하면 지정 이름으로 변경 이동

ex)
mv java /A/	현재 위치 java파일을 /A 아래 이동
mv java /A/jsp 	현재 위치 java파일을 /A 아래 jsp로 변경 이동

mv itbank google
- 현재 위치에서 재자리로 옮긴다
- 단, 파일명을 지정했기 때문에 이름이 변경되는 효과가 있다



※ Run Level
init 0		: 종료
init 1		: 싱글 부팅 (네트워크 x)
init 2		: 싱글 부팅 (여러계정 접속 가능, 네트워크 x)
init 3		: 멀티 부팅 (여러계정 접속 가능, 네트워크 o)
init 4		: 지정되지 않음
init 5		: XWindow (= GUI 모드)
init 6		: 재부팅
