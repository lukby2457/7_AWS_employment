vi : visual editor

1. 이동 모드
h : 좌측으로 한칸  이동
j : 아래로 한칸 이동
k : 위로 한칸 이동
l : 우측으로 한칸 이동

w : 다음 단어 첫 글자로 이동
e : 현재 단어 끝 문자로. 이미 끝이면 다음 단어 끝 문자로
b : 현재 단어 첫 문자로. 이미 처음이면 이전 단어 첫 문자로
x : 현재 커서 위치 문자 하나를 삭제


2. 잘라내기/복사/붙여넣기
dd : 현재 줄의 내용을 잘라낸다
yy : 현재 줄의 내용을 복사한다
p : 복사 및 잘라낸 니용을 커서 아래줄에 붙여넣는다
u : 되돌리기


3. 입력 모드
i : 현재 커서 앞부터 입력으로 진입
a : 현재 커서 뒤부터 입력으로 진입
o : 현재 커서 아래줄부터 입력으로 진입
s : 현재 커서의 문자 하나를 지우며 입력으로 진입
esc : 입력에서 이동모드로 빠져나온다


4. 실행모드 (:을 누르면 진입, esc로 빠져나옴)
:set nu - 줄번호를 출력
:w <파일명> - 지정이름으로 파일을 저장. 생략시 현재 파일에 저장
:q - 파일을 젖ㅇ하지 않고 닫는다. 수정시 실행되지 않음
:q! - 저장하지 않고 강제 종료
:wq - 저장 후 종료
